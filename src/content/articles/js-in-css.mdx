```html
<a href="/" class="my-nav-link">Главная</a>
<a href="/articles" class="my-nav-link current">Статьи</a>
```

```css
.my-nav-link {
  background-color: transparent;
  color: blue;
}

.my-nav-link.current {
  background-color: blue;
  color: white;
}
```

```html
<a href="/" class="my-nav-link">Главная</a>
<a href="/articles" class="my-nav-link" aria-current="page">Статьи</a>
```

```css
.my-nav-link {
  background-color: transparent;
  color: blue;
}

.my-nav-link[aria-current="primary"] {
  background-color: blue;
  color: white;
}
```
